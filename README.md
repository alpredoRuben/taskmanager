# APLIKASI WEB TASK MANAGER

## INSTALASI

> Clone Project Application
```
    git clone https://github.com/alpredoRuben/taskmanager.git

    atau 

    Download ZIP File
```

> Install Package Manager
```
    npm install

    or

    yarn add
```

> Jalankan Aplikasi Web nya
```
    npm start

    or

    yarn start
```

> Untuk build production
```
    npm run build

    or

    yarn build
```
